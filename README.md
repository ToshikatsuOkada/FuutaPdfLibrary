# FuutaPdfLibrary
## 合同会社ふうたシステムサービス 岡田年且

このライブラリは、PDF ファイルからテキストをを抽出するためのライブラリです。
基本的には「動けばよい」のスタンスで作成しているため、利用する際は、ライセンスに従い、ご自身で動作を確認の上ご利用ください。

また、本ライブラリは日本語にのみ対応しています。他言語については PDF の仕様上、正常に読み込めないケースが出てくる可能性が非常に高いのでご注意ください。
英文ぐらいならば大丈夫だとは思いますが、文字やフォントについて考慮が必要な文書に対しては、本ライブラリを修正して利用することも検討してください。

ライセンスは Apache 2.0 となります。ライセンスに従った利用については問題ありません。
また、良い修正が出来た場合は、プルリクを送ってください。とはいえ、余力が無いと送っていただいても対応などはかなり遅くなるとは思います。
