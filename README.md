# FuutaPdfLibrary : PDFのテキスト抽出用ライブラリ

## 合同会社ふうたシステムサービス 岡田年且

## 概要
このライブラリは、PDF ファイルからテキストをを抽出するためのライブラリです。
基本的には「動けばよい」のスタンスで作成しているため、利用する際は、ライセンスに従い、ご自身で動作を確認の上ご利用ください。

また、本ライブラリは日本語にのみ対応しています。他言語については PDF の仕様上、正常に読み込めないケースが出てくる可能性が非常に高いのでご注意ください。
英文ぐらいならば大丈夫だとは思いますが、文字やフォントについて考慮が必要な文書に対しては、本ライブラリを修正して利用することも検討してください。

ライセンスは Apache 2.0 となります。ライセンスに従った利用については問題ありません。
また、良い修正が出来た場合は、プルリクを送ってください。とはいえ、余力が無いと送っていただいても対応などはかなり遅くなるとは思います。develop ブランチ用意しておくので、プルリクを送るならばこっちにお願いします。

## サンプル
サンプルは、FuutaPdfLibrary プロジェクトにあります。
こちらのサンプルは、コンソールアプリとなっております。引数として PDF ファイルを指示すると、そのテキストを抽出して出力します。但し、あくまでもサンプルです。ですので、機能も最小限です。実際に使う場合は、これを参考にして組み込んでください。

## 注意点
このライブラリは、テキストだけ抽出します。なので、画像だけではなく数式とかも抽出できません。規格を読むと数式についても色々と書いてあるのですが、それはそれとして、どうやって出力するべきなのか？など決めないといけない点がいっぱいあります。また、このライブラリは、テキスト抽出を目的として開発したので、数式の対応は棚に上げてあります。
とはいえ、PDFフォーマット解析のベーシックな部分は書いてあるので、必要な方はうまいことライブラリを使って実装してください。自分で1から書くよりは楽だと思います。
